# Python for Oil and Gas

Python notebooks and exercises for working with oil and gas data. The repository focuses on applying Python to practical petroleum engineering and energy data analysis tasks.

## What This Covers

- Data handling with Python notebooks
- Petroleum and oil and gas datasets
- Basic analysis, visualisation and interpretation workflows
- Practice examples for building domain-focused Python skills

## How to Use

Open the notebooks in Jupyter Notebook, JupyterLab or Google Colab and run the cells in order. Review the code, outputs and comments to understand each workflow.

## Tools

Python, Jupyter Notebook, pandas, NumPy and matplotlib.
